Traitement de fichiers

gulp.src

L'instruction gulp.src permet de définir les fichiers à traiter et retourne un stream NodeJS.

Cette fonction prend en paramètre une chaîne de caractères accéptant les wildcards (*) :

gulp.src('stylesheets/*.css')

Il est également possible de définir un tableau contenant la liste des fichiers à traiter :

gulp.src(['bootstrap-*.css', 'font-awesome.css'])

gulp.pipe & gulp.dest

L'instruction .pipe de NodeJS permet ensuite de chaîner une liste de tâches.

Installez le package gulp-minify en utilisant NPM :

npm install --save-dev gulp-clean-css

Créez ensuite une tâche permettant de minimiser un fichier style.css et utilisez gulp.dest pour déterminer le fichier de destination :

const cleanCSS = require('gulp-clean-css')

gulp.task('css', function() {
  gulp.src('style.css')
  .pipe(cleanCSS())
  .pipe(gulp.dest('style.min.css'))
})
