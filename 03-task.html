Task

Créer une tâche

Gulp est un outil développer avec NodeJS permettant d'automatiser des tâches.

Pour créer une nouvelle tâche, il faut utiliser l'instruction gulp.task :

gulp.task('default', function() {
  console.log('Default task')
})

Exécuter une tâche

Vous pour maintenant exécuter cette tâche en utilisant la commande suivant depuis le terminal :

gulp default

La tâche étant nommée default, vous pouvez également l'exécuter avec la commande :

gulp

Dépendances

Il est également possible d'ajouter des dépendances lorsque vous exécutez une tâche.

Pour cela, ajouter en second paramètre un tableau contenant la liste des tâches à exécuter :

gulp.task('first', function() {
  console.log('First task')
})

gulp.task('second', ['first'], function() {
  console.log('Second task')
})
