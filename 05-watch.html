Watch

gulp.watch

L'instruction gulp.watch permet d'observer un ou plusieurs fichiers pour déclencher une tâche lorsque des modifications sont apportées.

Tout comme gulp.src, gulp.watch prend en premier paramètre une chaîne de caractères ou un tableau puis en second paramètre une fonction :

gulp.task('watch', function() {
  gulp.watch('js/**/*.js', function(event) {
    console.log('File ' + event.path + ' was ' + event.type + ', running tasks...')
  })
})

Cette tâche permet d'observer l'ensemble des fichiers JS contenus dans le dossier JS ou dans ses sous-dossiers.

Il est possible de définir un tableau de tâches en second paramètre
