Installation

Afin de pouvoir utiliser Gulp, il faut dans un premier temps installer NodeJS.

Utilisez ensuite le gestionnaire de packet NPM pour installer Gulp globalement :

npm install --global gulp

Initialisez ensuite votre projet NPM :

npm init

Installez ensuite la librairie Gulp localement dans votre projet :

npm install --save-dev gulp

Fichier gulpfile.js

Créez ensuite un fichier gulpfile.js. Ce fichier contiendra les tâches que nous souhaiterons éxecuter.

Vous pouvez dès maintenant ajouter l'appel à la librairie Gulp en haut du fichier :

const gulp  = require('gulp')
